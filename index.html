汇编眼中的函数

汇编眼中的函数

示例代码如下:

MOV ebx,1
MOV ecx,2

ADD ebx,ecx
MOV eax,ebx

说明:

实现两个数字相加，把相加的结果放入eax中
ADD ebx,ecx
MOV eax,ebx
这两行代码就相当于一个函数

下面来讲如何利用这两行代码的函数:

00FFFF01    MOV ebx,1
00FFFF02    MOV ecx,2
00FFFF03    CALL 00FFFF20
--------
00FFFF20    ADD ebx,ecx
00FFFF21    MOV eax,ebx
00FFFF22    RETN

01,02地址给ebx,ecx赋值
03地址CALL代码跳转到00FFFF20,此时的eip=00FFFF20,esp=00FFFF04
20地址将ebx,ecx相加结果在ebx中
21地址将ebx放入eax中
22地址rntn返回到04地址继续执行,eip=00FFFF03的值,esp=00FFFF03
